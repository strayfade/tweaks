# ShareClipboard

Now available on [strayfade's repo](https://strayfade.github.io/tweaks), tested with iOS 16.1.1 (and likely compatible with other versions)

### What is this?
Do you want to share your clipboard between a jailbroken iOS device and a Windows PC?
- Copy text/images on iOS, paste on Windows
- Copy text/images on Windows, paste on iOS

### Requirements
- ShareClipboard tweak from [strayfade's repo](https://strayfade.github.io/tweaks)
- ShareClipboard executable from [Releases](https://github.com/strayfade/ShareClipboard/releases/latest).
